3. 변수(Variable) - 반복적으로 사용되는 값을 변수로 지정
변수 이름 앞에 항상 $ 를 붙입니다.
$변수이름: 속성값

변수가 {} 안에서 선언 된 경우 {} 밖을 벗어나면 적용되지 않음 {} 안에 있는 변수를 사용하고 싶지만
{} 부모의 안에 포함되지 않는 요소의 경우 @at-root 를 쓰고 띄어쓰기 해주면 해당 부모를 벗어나게 할 수 있다.
* 변수 유효범위 (Scope)
변수는 사용가능한 유효범위가 있다.
선언된 블록({}) 내에서만 유효범위를 가진다.

*전역설정 !global
!global 플래그를 사용하면 변수의 유효범위를 전역(Global)로 설정할 수 있다.(잘 쓰지 않음. 보기 불편하니까)

4. 재활용 (mixins) !중요!
스타일시트 전체에서 재사용 할 css선언 그룹을 정의해줌
1) 선언하기(@mixin)
@mixin 믹스인 이름 {
	스타일;
}
@mixin믹스인이름($매개변수) {
	스타일;
}
@mixin large-text {
	font-size: 22px;
	font-weight: bold;
	font-family: sans-serif;
	color: tomato;
	&::after {
		content:"여러분";
	}
}
div {
	@include large-text;
}
2) 사용(포함)하기(!include)
@include 믹스인 이름;
@include 믹스인이름(인수);

@mixin dash-line($width: 1px, $color: black) {
    border: $width dashed $color
}
